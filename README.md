# Scraping API Project

Данный проект предоставляет RESTful API для взаимодействия с данными, собранными на предыдущем этапе проекта (скрапинг).
API разработано с использованием фреймворка FastAPI, что обеспечивает высокую производительность, строгую валидацию запросов и автоматическую документацию.

#Структура проекта

* ```main.py``` - Основной код приложения FastAPI (эндпоинты, логика фильтрации и пагинации).

* ```requirements.txt``` - Список зависимостей.

* ```data/sample_data.csv``` - Небольшой срез сырых данных (база), используемый для демонстрации работы API.

* ```README.md``` - Инструкция по запуску и работе с репозиторием.

# Особенности реализации

1. Пагинация: Реализованы параметры ```skip``` и ```limit```. Параметр ```limit``` жестко ограничен (максимум 50 записей за запрос), что предотвращает утечку всего дампа базы данных и экономит трафик.

2. Фильтрация: Можно фильтровать данные по минимальной цене (```min_price```), максимальной цене (```max_price```) и наличию (```availability```).

3. Сортировка: Доступна сортировка по цене, дате добавления и названию, как по возрастанию, так и по убыванию.

4. Валидация: FastAPI автоматически проверяет типы данных (например, если передать строку вместо числа в ```limit```, API вернет понятную 422 ошибку).

# Быстрый старт и запуск GUI (Документации)

1. Установка зависимостей:
В терминале, находясь в корневой папке проекта, выполните:
```
python -m venv venv
source venv/bin/activate  # Для Windows: venv\Scripts\activate
pip install -r requirements.txt
```


2. Запуск сервера API:
```
uvicorn main:app --reload
```

Параметр ```--reload``` позволяет серверу автоматически перезапускаться при изменении кода (полезно при разработке).

3. Как открыть интерфейс (Swagger GUI) и варианты использования:

Откройте в браузере ссылку: https://www.google.com/search?q=http://127.0.0.1:8000/docs

Это автоматически сгенерированная документация. Здесь вы можете протестировать API прямо из браузера (кнопка ```Try it out```):

* Вариант 1: Получить первые 5 книг: Установите ```limit``` = 5, остальные поля пустые.

* Вариант 2: Найти книги дешевле 40£: Установите ```max_price``` = 40.
